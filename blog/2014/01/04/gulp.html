Gulp - First impressions

04 Jan 2014

In the last week of 2013, on my Twitter feed (or it was on Google Plus? Doesn't matter), I saw a link about something called Gulp. I opened up that link, scrolled the page like a ninja and didn't understand anything, of course, because I didn't read the article...

Gulp is a Javascript task runner/streaming build system created by Fractal. Gulp uses the concept about Streams.

So, Today I've started to play with Gulp and I wanna share with you my first impressions.

If you'd like to learn how to get start, I highly recomend the documentation and this article.

What I liked:

Simplicity:Gulp is code over configuration, you don't need to care much about how to configure your environment, in fact, you basically need to know 5 functions:

// Return a readable stream
gulp.src(glob)

// Return a writable stream
gulp.dest(folder)

// Register a task
gulp.task(name, fn)

// Execute tasks
gulp.run(tasks)

// Watch file changes
gulp.watch(glob, fn)

Bellow an example of a real gulpfile.js:

// Include gulp
var gulp = require('gulp');

// Plugins
var compass = require('gulp-compass');

// Tasks

// Compass
gulp.task('compass', function() {
  gulp.src('./assets/scss/*.scss')
      .pipe(compass({
        config_file: './config.rb'
      }))
      .pipe(gulp.dest('./assets/css'));
});

// Default Task
gulp.task('default', function() {
  gulp.run('compass');

  // Watch the files
  gulp.watch('./assets/scss/**/*.scss', function() {
    gulp.run('compass');
  });
});

Pretty simple, I really liked it.

What I dislike

Concurrency model:As far as I understood, every task runs in parallel, in other words, you can't have a task that execute other tasks in order. Maybe I'm wrong, at least I didn't figured out how can I do this.

Final Words

To finish up, I think Gulp is a great tool, already have more than 100 plugins and this number will surely grows. For now, I'll still using Grunt as my task runner of choice, but, I'll keep my eyes open on Gulp, because is very good to know that you have other choices!

That's all folks!
