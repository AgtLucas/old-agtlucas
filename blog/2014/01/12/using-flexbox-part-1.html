Using Flexbox - Part I

12 Jan 2014

Yesterday I talked a little about CSS Flexible Box Layout Model, and today I'll show you how to use Flexbox (I'll try).

Take a look at the image below:

To achieve that with flexbox, you should do something like this:

.flex-container {
  display: flex;
}

.flex-item {
  background: gray;
  height: 400px;
  margin: 5px;
}

.mini-box {
  width: 200px;
}

.mega-box {

}
