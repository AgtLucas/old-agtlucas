Clearfix Sass Mixin

14 Feb 2014

If you're like me and use Sass, probably you know the mixin above, but if you don't, no problem.

In 2011, Nicolas Gallagher present us with his clearfix hack approach, since then this hack has getting very, very popular, and been present in almost every site that I visit every day.

Here's the mixin:

@mixin cf() {
  &:after,
  &:before {
      content: "";
      display: table;
  }
  &:after {
      clear: both;
  }
}

Usage:

.foo {
  @include cf();
}

Simple as had to be!
